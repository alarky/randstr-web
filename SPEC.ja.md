# Random Strings - UI・機能仕様書

## 概要

ランダム文字列を生成する静的 Web アプリケーション。ユーザーが文字種・個数・長さを指定し、暗号論的に安全な乱数で文字列を生成する。

### 技術構成

- HTML + CSS + JavaScript (ビルド不要)
- Alpine.js (リアクティブ UI)
- Pico.css (ベーススタイル、ダークテーマ)
- フォント: `system-ui, sans-serif` (UI部)、`Menlo, Consolas, 'Courier New', monospace` (結果表示部)

## UI 要素

### 入力フィールド

| 要素 | 種類 | デフォルト値 | 説明 |
|------|------|-------------|------|
| count | text input | `12` | 生成する文字列の個数 |
| length | text input | `16` | 各文字列の長さ |

- 入力値は `parseInt` で整数に変換、パース不可時は `0`

### ボタン

| ボタン | 動作 |
|--------|------|
| Generate | 現在の設定で文字列を再生成 |
| Reset | URL パラメータをクリアし、全設定をデフォルトに戻す |

### 文字種チェックボックス（基本）

| ラベル | デフォルト | 文字セット |
|--------|-----------|-----------|
| a-z | ON | `abcdefghijklmnopqrstuvwxyz` |
| A-Z | ON | `ABCDEFGHIJKLMNOPQRSTUVWXYZ` |
| 0-9 | ON | `0123456789` |
| symbols | OFF | 全記号の一括トグル（後述） |

### 記号の出現割合制限 (symbol max)

- 記号が1つ以上 ON のときのみ表示される
- デフォルト: `20` (%)
- 生成される文字列中の記号文字数が `length * symbolMax / 100` を超えないよう制限
- 英数字のみ or 記号のみの場合は制限なし

### 記号チェックボックス（個別, 全33種）

デフォルトは全て OFF。2行に分かれて表示される。

**1行目 (15個):**
`!` `"` `#` `$` `%` `&` `'` `(` `)` `*` `+` `,` `-` `.` `/`

**2行目 (18個):**
`:` `;` `<` `=` `>` `?` `@` `[` `\` `]` `^` `_` `` ` `` `{` `|` `}` `~`

### 結果表示

- `<ul>` / `<li>` のリスト形式で表示（左ボーダー付きカード風）
- 文字列数は count の値に従う
- クリックでクリップボードにコピー、「Copied!」ミニトースト表示

## 機能仕様

### 文字列生成

1. ON になっている文字種の文字を英数字プールと記号プールに分けて構築する
2. 両プールが空の場合、結果は空配列（何も表示しない）
3. 片方のプールのみの場合、そのプールから均等にランダム選択
4. 両方のプールがある場合、記号の出現数が `symbolMax` の割合を超えないよう制限しつつ生成
5. 乱数は `crypto.getRandomValues()` (Web Crypto API) を使用
   - `Uint32Array` を取得し、各値を文字プール長で割って文字を選択

### リアクティブ動作

- count / length の変更 → 即座に再生成
- a-z / A-Z / 0-9 チェックボックスの変更 → 即座に再生成
- 個別記号チェックボックスの変更 → 即座に再生成
- symbol max の変更 → 即座に再生成
- Generate ボタン → 現在の設定で再生成
- 初回表示時 → 自動的に生成

### symbols 一括トグル

- **symbols ON**: 全33個の記号チェックボックスを ON にして再生成
- **symbols OFF**: 全33個の記号チェックボックスを OFF にして再生成
- **個別記号の変更時**: 全記号の状態が一致したら symbols チェックボックスもその状態に同期

### クリップボードコピー

- 結果の `<li>` をクリックすると `navigator.clipboard.writeText()` でコピー
- クリックした行の左ボーダーが緑に変化（1秒後に復帰）
- 行の右端に「Copied!」トーストが表示（0.8秒でフェードアウト）

### Reset

- URL のクエリパラメータをクリアする (`window.history.replaceState`)
- 全設定をデフォルト値に復元

## URL パラメータによる状態永続化

設定変更時に URL クエリパラメータを `history.replaceState` で更新する。ページ読み込み時にパラメータから状態を復元する。

### パラメータ

| キー | 説明 | 例 |
|------|------|----|
| `c` | count (文字列の個数) | `c=20` |
| `l` | length (文字列の長さ) | `l=32` |
| `ch` | 有効な文字種トークン | `ch=aA0` |
| `sym` | 有効な記号 | `sym=all` or `sym=!%23%24` |
| `sm` | symbol max % (デフォルト時は省略) | `sm=10` |

### `ch` トークン

| トークン | 文字種 |
|----------|--------|
| `a` | 小文字 (a-z) |
| `A` | 大文字 (A-Z) |
| `0` | 数字 (0-9) |

例: `ch=aA0` (全て有効)、`ch=a` (小文字のみ)、`ch=A0` (大文字+数字)

### `sym` 値

| 値 | 意味 |
|----|------|
| 省略 | 全記号 OFF |
| `all` | 全33記号 ON |
| 個別文字の連結 | 指定した記号のみ ON (例: `sym=!@#`) |

### デフォルト値（URL パラメータなし時）

- count: `12`
- length: `16`
- ch: `aA0` (a-z: ON, A-Z: ON, 0-9: ON)
- sym: 省略 (全記号 OFF)
- sm: `20`
