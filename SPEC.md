# Random Strings - UI・機能仕様書

## 概要

ランダム文字列を生成する Web アプリケーション。ユーザーが文字種・個数・長さを指定し、暗号論的に安全な乱数で文字列を生成する。

## UI 要素

### 入力フィールド

| 要素 | 種類 | デフォルト値 | 説明 |
|------|------|-------------|------|
| count | text input | `12` | 生成する文字列の個数 |
| length | text input | `16` | 各文字列の長さ |

- 入力値は `parseInt` で整数に変換、パース不可時は `0`

### ボタン

| ボタン | 動作 |
|--------|------|
| Generate | 現在の設定で文字列を再生成 |
| Reset | URL パラメータをクリアし、全設定をデフォルトに戻す |

### 文字種チェックボックス（基本）

| ラベル | デフォルト | 文字セット |
|--------|-----------|-----------|
| a-z | ON | `abcdefghijklmnopqrstuvwxyz` |
| A-Z | ON | `ABCDEFGHIJKLMNOPQRSTUVWXYZ` |
| 0-9 | ON | `0123456789` |
| symbols | OFF | 全記号の一括トグル（後述） |

### 記号チェックボックス（個別, 全33種）

デフォルトは全て OFF。2行に分かれて表示される。

**1行目 (15個):**
`!` `"` `#` `$` `%` `&` `'` `(` `)` `*` `+` `,` `-` `.` `/`

**2行目 (18個):**
`:` `;` `<` `=` `>` `?` `@` `[` `\` `]` `^` `_` `` ` `` `{` `|` `}` `~`

### 結果表示

- `<ul>` / `<li>` のリスト形式で表示
- 文字列数は count の値に従う

## 機能仕様

### 文字列生成

1. ON になっている文字種の文字を全て結合して文字プール（候補文字列）を作る
2. 文字プールが空の場合、結果は空配列（何も表示しない）
3. `count` 回ループし、各ループで `length` 文字のランダム文字列を生成
4. 乱数は `window.crypto.getRandomValues()` (Web Crypto API) を使用
   - `Uint32Array(length)` を一括取得し、各値を文字プール長で割って文字を選択

### リアクティブ動作

- count / length の変更 → 即座に再生成
- a-z / A-Z / 0-9 チェックボックスの変更 → 即座に再生成
- 個別記号チェックボックスの変更 → 即座に再生成
- Generate ボタン → 現在の設定で再生成
- 初回表示時 → 自動的に生成

### symbols 一括トグル

- **symbols ON**: 全33個の記号チェックボックスを ON にして再生成
- **symbols OFF**: 全33個の記号チェックボックスを OFF にして再生成
- **個別記号の変更時**: 全記号の状態が一致したら symbols チェックボックスもその状態に同期

### Reset

- URL を `/` に戻す (`window.history.replaceState`)
- 全設定をデフォルト値に復元

## URL パラメータによる状態永続化

設定変更時に URL クエリパラメータを `history.replaceState` で更新する。ページ読み込み時にパラメータから状態を復元する。

### パラメータ

| キー | 説明 | 例 |
|------|------|----|
| `c` | count (文字列の個数) | `c=20` |
| `l` | length (文字列の長さ) | `l=32` |
| `f` | flags (36桁の0/1文字列) | `f=111000000000000000000000000000000000` |

### flags (`f`) のビット順序

```
位置  0: a-z (useLowers)
位置  1: A-Z (useUppers)
位置  2: 0-9 (useDigits)
位置  3: symbols 一括フラグ (useSymbols)
位置  4: !    位置  5: "    位置  6: #    位置  7: $
位置  8: %    位置  9: &    位置 10: '    位置 11: (
位置 12: )    位置 13: *    位置 14: +    位置 15: ,
位置 16: -    位置 17: .    位置 18: /    位置 19: :
位置 20: ;    位置 21: <    位置 22: =    位置 23: >
位置 24: ?    位置 25: @    位置 26: [    位置 27: \
位置 28: ]    位置 29: ^    位置 30: _    位置 31: `
位置 32: {    位置 33: |    位置 34: }    位置 35: ~
```

### デフォルト値（URL パラメータなし時）

- count: `12`
- length: `16`
- a-z: ON, A-Z: ON, 0-9: ON
- symbols および全個別記号: OFF
